# mystore

Demo project: Vue 3 + Vite + Tailwind CSS + Firebase (Firestore).

## Setup

1. Install deps:
   npm install

2. Configure Firebase:
   - Edit src/firebase.js and replace the firebaseConfig placeholders with your project's config.

3. Run dev server:
   npm run dev

4. Build:
   npm run build

5. Deploy (example with Surge):
   npm run build
   npx surge dist/ your-domain.surge.sh
